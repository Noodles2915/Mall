# 智能商城系统

**OUC-ITstudio 2026寒假作业项目**

**Developed by 海大睡觉队**

## 基础信息

本项目是一个前后端分离的电商平台系统，包含用户认证、商品管理、购物车、订单系统等核心功能。

- **后端**：Django + DRF + MySQL
- **前端**：Vue 3
- **核心功能**：用户管理、商品展示、购物车、订单流程、后台管理

### 功能实现&特点

**后端**：多权限级别的用户管理；支持默认设置的地址管；，商家管理员均可使用，且具有两级鉴权的商品管理系统；订单交易系统；基于Django Admin二次开发的管理系统

**前端**：商品展示；用户界面；基于用户权限组改变的导航
