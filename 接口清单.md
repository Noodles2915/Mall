# 商城系统接口清单(前后端对接用)

## 通用说明

- 基础路径：`/api`
- 鉴权方式：JWT
- 需要登录的接口，请在请求头携带：

```http
Authorization: Bearer <access_token>
```

- 统一返回结构：

```json
{
	"code": 0,
	"message": "ok",
	"data": {}
}
```

## 用户和地址系统

### 1. 注册

- 接口：`POST /api/auth/register/`

请求示例：

```json
{
	"username": "alice",
	"email": "alice@example.com",
	"password": "123456"
}
```

返回示例：

```json
{
	"code": 0,
	"message": "ok",
	"data": {
		"access": "<access_token>",
		"refresh": "<refresh_token>",
		"user": {
			"id": 1,
			"username": "alice",
			"email": "alice@example.com",
			"avatar": null
		}
	}
}
```

### 3. 商品列表

- 接口：`GET /api/products/`
- 筛选参数：
- `keyword`：关键词，匹配商品名称/副标题/描述
- `category`：分类ID，支持父级分类（会同时包含其子分类商品）
- `is_hot`：是否仅看热门（`1/true/yes`）

请求示例：

```http
GET /api/products/?keyword=手机&category=1&is_hot=true
```

返回示例：

```json
{
	"code": 0,
	"message": "ok",
	"data": [
		{
			"id": 1,
			"name": "Aster X1 Pro 手机",
```
