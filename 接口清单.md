# 商城系统接口清单(前后端对接用)

## 用户和地址系统

**JWT对接：在请求头使用`Authorization: Bearer <access_token>`。**

- `POST /api/auth/register/` : 注册并返回`access/refresh`
- `POST /api/auth/login/` : 登录并返回`access/refresh`
- `GET /api/auth/me/` : 当前用户信息
- `PUT /api/auth/me/` : 更新当前用户信息
- `GET /api/auth/addresses/` : 地址列表
- `POST /api/auth/addresses/` : 新建地址
- `GET /api/auth/addresses/<pk>/` : 地址详情
- `PUT/PATCH /api/auth/addresses/<pk>/` 更新地址
- `DELETE /api/auth/addresses/<pk>/` 删除地址
- `POST /api/auth/addresses/<pk>/default` 设为默认地址
